<decisions_made>

- **Vercel-only (no Railway):** Requires rename api/ → backend/ (Vercel treats api/ as zero-config functions dir), /api prefix on all routes, and api/index.py as the Vercel entry point wrapping the FastAPI app via mount.
- **api/index.py mount wrapper:** `app.mount("/api", _backend_app)` — Vercel zero-config routes /api/* to this function; mount strips /api before delegating to backend.main:app so all internal routes (tests, uvicorn dev) stay prefix-free.
- **Vite proxy path-stripping:** `/api` proxy entry with `rewrite: path => path.replace(/^\/api/, '')` — dev backend still runs with `uvicorn backend.main:app` (no /api prefix needed).
- **Test files excluded from TS build:** tsconfig.app.json `"exclude": ["src/tests"]` — Vercel's tsc -b step was failing on test-only type assertions incompatible with erasableSyntaxOnly.
