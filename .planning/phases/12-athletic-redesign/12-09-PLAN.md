Do NOT add `--color-muted-foreground` or any other token — CardDescription/`text-muted-foreground` is not used by SettingsScreen and is out of scope for this gap.

2. In `frontend/src/components/ui/card.tsx`, in the `Card` function's className string (currently `flex flex-col gap-6 rounded-xl border bg-card py-6 text-card-foreground shadow-sm`), add the `border-border` utility next to the existing `border` utility so the Card's border-color resolves to `--color-border` (already defined in index.css as `var(--color-line)` = #DFE0E2 = rgb(223,224,226)). `border` keeps the 1px width; `border-border` sets the color.
