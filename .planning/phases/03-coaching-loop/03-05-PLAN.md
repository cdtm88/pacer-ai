    - `async def detect_signals(user_id: str, window_days: int = 7) -> list[dict]`: implements D-16/D-17. Missed-session check: load planned sessions whose scheduled_date is past due within the window and have status 'planned'; for each, look for a matching ride within ±1 day of the scheduled_date; if none, append {"type": "missed", "session_id": id}. Underperformance check: load recent rides in the window; for each ride matched to a planned session, call validate_session_vs_actual(planned={"tss": planned_tss}, actual={"tss": ride_tss}); if the returned compliance_pct is not None and < 60, append {"type": "underperformance", "session_id": id, "compliance_pct": value}. Count sessions whose scheduled_date moved by more than 1 day between before and after; compute shift_pct = shifted / total (0 when total is 0); return {"shifted_count", "total_upcoming", "shift_pct", "requires_user_confirmation": shift_pct > 0.30}. scheduled_date values may be ISO strings or date objects — parse robustly before diffing.
