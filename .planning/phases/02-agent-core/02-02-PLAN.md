<objective>
Build the transport-agnostic agent core: the tool registry + dispatcher (`agent/tools.py`) that wraps Phase 1's 8 sports_science functions as manual Anthropic tool schemas (D-03, D-04, D-05, TRUST-02), and the multi-turn agentic loop (`agent/loop.py`) that uses the raw anthropic SDK with an explicit `stop_reason == "tool_use"` check, deduplicates by `(name, args_hash)`, dispatches parallel tool calls via `asyncio.gather`, enforces a 3-retry cap, and surfaces tool failures (AGENT-01, AGENT-02, AGENT-03, AGENT-04, TRUST-04, TRUST-05).

Output: `agent/tools.py` (TOOL_SCHEMAS, TOOL_REGISTRY, dedup_key, dispatch_tool) and `agent/loop.py` (run_turn async generator yielding SSE-shaped event dicts).
</objective>

Loop while `retries <= MAX_RETRIES`. Inside each iteration open `async with client.messages.stream(model=model, max_tokens=4096, tools=TOOL_SCHEMAS, messages=messages) as stream:`; collect text deltas in an `async for event in stream` loop, buffering `event.delta.text` when present and yielding a `token` event for each delta; after the async-for completes, call `final_msg = await stream.get_final_message()` (Pitfall 3: never await get_final_message inside the iteration).
