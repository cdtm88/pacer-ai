<objective>
Build the two isolated browser hooks that power the during-session screen: useSessionTimer (Date.now() delta countdown with visibilitychange resync) and useWakeLock (native Wake Lock with NoSleep.js fallback). Install nosleep.js and add its TypeScript declaration. These hooks have no dependency on the screen or the backend, so they ship in parallel with the backend.

Purpose: iOS Safari throttles background timers and dropped wake lock in installed PWAs before 18.4; both behaviors require the exact patterns these hooks encapsulate (IOS-01, IOS-02). Isolating them as testable hooks keeps the screen rewrite (plan 03) thin.
Output: nosleep.js dependency, frontend/src/types/nosleep.d.ts, frontend/src/hooks/useSessionTimer.ts, frontend/src/hooks/useWakeLock.ts, and unit tests for both.
</objective>
