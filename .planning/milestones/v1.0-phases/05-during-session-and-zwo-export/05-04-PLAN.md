## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-11 | Spoofing | export.zwo request | mitigate | apiFetch injects the Supabase JWT; the backend dual-filters by user_id (plan 01) so a user can only download their own session |
| T-05-12 | Information disclosure | error toast text | mitigate | Toast copy is generic ("Export failed...", "Session not found...") and never leaks server internals or stack traces |
| T-05-13 | Denial of service | popup-blocked download | mitigate | Use URL.createObjectURL + hidden anchor click (not window.open) so popup blockers do not break the download |
| T-05-SC | Tampering | npm installs | mitigate | No new package installs in this plan; nosleep.js is installed in plan 02 |
</threat_model>
